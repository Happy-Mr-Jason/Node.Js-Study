## Cookies

Storage in Browser. Tracking the activities of Client.

Module for cookie

refer to [npm cookie-parser](https://www.npmjs.com/package/cookie-parser)

Installation

```command
npm install cookie-parser
```

Signed Cookie

```javascript
app.use(cookieParser('SECRET_KEY'));
```
